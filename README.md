# Spring Boot アプリ集

Spring Bootの学習として複数のWebアプリを作成しました。


## ① Todoアプリ

### 概要
タスクの管理（追加・編集・削除・完了管理）ができるWebアプリです。
もともと名前管理アプリとして作成したシンプルなCRUDアプリをベースに、
Todoアプリとして段階的に機能拡張を行い開発しました。

### 機能
・タスクの追加
・タスクの編集
・タスクの削除
・タスクの完了/未完了切り替え

### 画面
・一覧画面 (タスクの表示・削除・編集)
・編集画面 (タスクの更新)

### 使用技術
・Java
・Spring Boot
・H2 Database
・Thymeleaf

### 処理の流れ (例：更新機能)
編集ボタン押下時、ControllerでIDを受け取り、
Serviceを通してDBから対象データを取得し編集画面に表示します。

更新時は、入力値をControllerで受け取り、
ServiceでDBを更新後、一覧画面へリダイレクトします。

### 工夫した点
・完了したタスクは見た目（取り消し線・色）を変更
・Flexboxを使用してボタン配置を調整し、見やすさを向上

### 今後の課題
・バリデーション機能の実装（現在学習中）
・検索・フィルター機能の追加


## ② ユーザー管理アプリ

### 概要
ユーザー登録・ログイン・ログアウト機能を実装したWebアプリです。
セッションを用いてログイン状態を管理しています。

### 機能
・ユーザー登録
・ログイン/ログアウト
・セッション管理
・ユーザー一覧表示

### 画面
・登録画面
・ログイン画面
・ユーザー一覧画面

### 使用技術
・Java
・Spring Boot
・H2 Database
・Thymeleaf

### 処理の流れ（例：ログイン）
ログイン時、Controllerで入力値を受け取り、
Serviceでユーザー情報を取得して認証を行います。

認証成功時はセッションにユーザー情報を保持し一覧画面へ遷移、
未ログイン時は一覧画面へアクセスできないよう制御しています。

### 工夫した点
・セッションを利用してログイン状態を保持
・未ログイン時は一覧画面にアクセスできないよう制御

### 今後の課題
・パスワードのハッシュ化
・Spring Securityの導入
