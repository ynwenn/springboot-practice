# Spring Boot アプリ集

Spring Bootの学習として複数のWebアプリを作成しました。  
本リポジトリには以下のアプリがあります。

## ポートフォリオ一覧
① Todoリスト
TodoController / TodoService / TodoRepository などで構成しています。  
② ユーザー管理アプリ
UserController / UserService / UserRepository などで構成しています。  
③ 掲示板アプリ
PostController / PostService / PostRepository などで構成しています。  

※ 各アプリの詳細は下記に記載しています。


## ① Todoアプリ

### 概要
タスクの管理（追加・編集・削除・完了管理）ができるWebアプリです。  
もともと名前管理アプリとして作成したシンプルなCRUDアプリをベースに、
Todoアプリとして段階的に機能拡張を行い開発しました。

### 機能
・タスクの追加  
・タスクの編集  
・タスクの削除  
・タスクの完了/未完了切り替え

### 画面
・一覧画面 (タスクの表示・削除・編集)  
・編集画面 (タスクの更新)

### 使用技術
・Java  
・Spring Boot  
・H2 Database  
・Thymeleaf

### 処理の流れ (例：更新機能)
編集ボタン押下時、ControllerでIDを受け取り、  
Serviceを通してDBから対象データを取得し編集画面に表示します。  
  
更新時は、入力値をControllerで受け取り、  
ServiceでDBを更新後、一覧画面へリダイレクトします。

### 工夫した点
・完了したタスクは見た目（取り消し線・色）を変更  
・Flexboxを使用してボタン配置を調整し、見やすさを向上

### 今後の課題
・バリデーション機能の実装（現在学習中）  
・検索・フィルター機能の追加


## ② ユーザー管理アプリ

### 概要
ユーザー登録・ログイン・ログアウト機能を実装したWebアプリです。  
セッションを用いてログイン状態を管理しています。

### 機能
・ユーザー登録  
・ログイン/ログアウト  
・セッション管理  
・ユーザー一覧表示

### 画面
・登録画面  
・ログイン画面  
・ユーザー一覧画面

### 使用技術
・Java  
・Spring Boot  
・H2 Database  
・Thymeleaf

### 処理の流れ（例：ログイン）
ログイン時、Controllerで入力値を受け取り、  
Serviceでユーザー情報を取得して認証を行います。  
  
認証成功時はセッションにユーザー情報を保持し一覧画面へ遷移、  
未ログイン時は一覧画面へアクセスできないよう制御しています。

### 工夫した点
・セッションを利用してログイン状態を保持  
・未ログイン時は一覧画面にアクセスできないよう制御

### 今後の課題
・パスワードのハッシュ化  
・Spring Securityの導入


## ③ 掲示板アプリ

### 概要
ユーザーが投稿を行い、一覧で閲覧できる掲示板アプリです。  
②のログイン機能と連携し、投稿者の管理を行っています。

### 機能
・投稿の作成  
・投稿一覧表示  
・投稿の削除  
・投稿とユーザーの紐づけ  
・ログインユーザーのみ削除可能な制御

### 画面
・投稿一覧画面

### 使用技術
・Java  
・Spring Boot  
・H2 Database  
・Thymeleaf

### 処理の流れ（例：投稿）
ログイン中のユーザー情報をセッションから取得し、  
投稿データに紐づけて保存しています。  
  
一覧表示時には投稿内容と投稿者を表示し、  
ログインユーザーと投稿者が一致する場合のみ削除操作を可能としています。

### 工夫した点
・ユーザーと投稿のリレーションを実装  
・ログインユーザーと投稿者を比較し、操作制御を実現  
・Flexboxを用いてレイアウトを調整し、見やすさを向上

### 今後の課題
・バリデーションの追加  
・ページネーション機能
